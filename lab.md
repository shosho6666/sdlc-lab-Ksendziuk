1. Планування

Мета продукту:
Користувач повинен мати можливість вводити назви страв та їхню вагу (у грамах), щоб отримувати розрахунок калорійності кожної страви та загальної кількості калорій за день.

Основна ідея застосунку — допомогти користувачам контролювати свій раціон харчування та слідкувати за добовою нормою калорій.

2. Аналіз вимог (User Stories)

Як користувач, я хочу додавати страву із зазначенням кількості грамів, щоб розрахувати її калорійність. ✅ (Must have)

Як користувач, я хочу бачити загальну кількість калорій за день, щоб контролювати своє харчування. ✅ (Must have)

Як користувач, я хочу видаляти страву зі списку, щоб виправляти помилки у введених даних.

Як користувач, я хочу бачити список усіх доданих страв в одному місці, щоб аналізувати свій раціон.

Як користувач, я хочу очищати список страв за день, щоб почати підрахунок заново.

function addMeal(name, grams):

    if name is empty:
        return "Error: Name cannot be empty"

    if grams <= 0:
        return "Error: Invalid weight"

    caloriesPer100g = getCaloriesFromDatabase(name)

    totalCalories = (caloriesPer100g / 100) * grams

    meal = createMeal(name, grams, totalCalories)

    mealList.add(meal)

    updateTotalCalories()

    return "Meal added successfully"
